# HUSTmotor 多模态电机故障数据集说明

> HUSTmotor Multimodal Motor Fault Dataset Documentation

数据来源：<https://github.com/CHAOZHAO-1/HUSTmotor-multi-modal-dataset>

## 1. 数据集概述

HUSTmotor 数据集是由湖南科技大学（HUST）发布的电机故障诊断公开数据集，包含电机在 **4 种运行转速**下、**6 种健康状态**下的振动信号（X/Y/Z 三轴加速度计）和声学信号（Sound 麦克风），广泛用于验证各类故障诊断算法。

### 1.1 实验平台

故障模拟实验使用 Spectra-Quest 机械故障模拟器（Mechanical Fault Simulator），可精确模拟多种电机典型故障。

### 1.2 传感器配置

| 通道 | 传感器类型 | 说明 |
|------|-----------|------|
| X | 加速度计 | 径向振动（方向1） |
| Y | 加速度计 | 径向振动（方向2） |
| Z | 加速度计 | 轴向振动（方向3） |
| Sound | 麦克风 | 声学信号 |

### 1.3 采集参数

| 参数 | 值 |
|------|-----|
| 采样频率 | 25.6 kHz（25,600 Hz） |
| 每次采样点数 | 163,840 点 |
| 采样时长 | ~6.4 秒 |
| 采样间隔 | 39.06 µs |
| 数据块数 | 5 块（每块含完整采样周期） |

## 2. 故障类型（6 种状态）

| 标签 | 全称 | 含义 |
|------|------|------|
| **H** | Healthy | 无故障，正常运行的基准数据 |
| **BF** | Bearing Fault | 滚动轴承故障（内圈/外圈/滚动体） |
| **BOW** | Bowed Rotor | 转子弯曲，导致不平衡振动 |
| **BR** | Broken Rotor Bars | 转子导条断裂（异步电机典型故障） |
| **MIS** | Rotor Misalignment | 转轴与负载对中不良 |
| **UV** | Voltage Unbalance | 三相供电电压不对称 |

> 注：原始数据集中文件名前缀为 `BROKEN`、`MISAL`、`UNBAL`，项目中通过 `config.py` 的 `LABEL_MAP` 映射为标准短标签 `BR`、`MIS`、`UV`。

## 3. 运行工况（4 种转速）

| 转速频率 | 说明 |
|---------|------|
| 5 Hz | 低速 |
| 10 Hz | 中低速 |
| 20 Hz | 中高速 |
| 30 Hz | 高速 |

## 4. 文件命名规则

格式：**`[状态编码]_[转速]HZ.txt`**

示例：

- `H_5HZ.txt` — 健康电机，5 Hz 转速
- `BF_10HZ.txt` — 轴承故障，10 Hz 转速
- `BOW_20HZ.txt` — 弯曲转子，20 Hz 转速
- `BROKEN_30HZ.txt` — 转子断条，30 Hz 转速

共 **6（状态）× 4（转速）= 24 个文件**。

## 5. 文件内部结构

### 5.1 文件头部（元数据，前 19 行）

每个 `.txt` 文件开头包含实验参数说明：

```text
Title: H_5HZ
Parameters:

DAQ Settings:
Frequency Limit    10000.000000
Spectral Lines     12800
Number of Blocks   5
Total Data Rows    163840
Channels:
Legend      X       Y       Z       Sound
Tacho1
On/Off      ON      ON      ON      ON
OFF

Volts/Unit  1.000000 1.000000 1.000000 1.000000
```

关键字段说明：

| 字段 | 含义 |
|------|------|
| Frequency Limit | 抗混叠滤波器截止频率（10,000 Hz） |
| Spectral Lines | 频谱分析线数（12,800 线） |
| Number of Blocks | 数据块数（5 块） |
| Total Data Rows | 总数据行数（163,840） |
| On/Off | 通道开关状态 |
| Volts/Unit | 电压-物理量转换系数 |

### 5.2 数据体（制表符分隔）

头部之后以 `Time (seconds) and Data Channels` 标记数据起始，随后为 Tab 分隔的数值矩阵：

```text
Time (seconds) and Data Channels
0.00000000  -0.01332358  -0.01647493  -0.00948236  -0.01588489
0.00003906  -0.01498642  -0.01854156  -0.01159816  -0.01590575
0.00007813  -0.01472269  -0.01870695  -0.01168011  -0.01581039
...（共 163,840 行）
```

| 列 | 名称 | 单位 | 说明 |
|----|------|------|------|
| 1 | Time | 秒 | 采样时间戳（从 0 开始） |
| 2 | X | 电压 | X 轴振动信号 |
| 3 | Y | 电压 | Y 轴振动信号 |
| 4 | Z | 电压 | Z 轴振动信号 |
| 5 | Sound | 电压 | 声学信号 |

### 5.3 读取方式

使用 pandas 读取时需跳过头部 19 行，以 Tab 分隔：

```python
import pandas as pd
df = pd.read_csv(
    "BF_10HZ.txt",
    skiprows=19,
    sep="\t",
    header=None,
    names=["Time", "X", "Y", "Z", "Sound"],
    encoding="utf-8",
    engine="python",
)
```

## 6. 数据集规模

| 指标 | 数值 |
|------|------|
| 文件总数 | 24 个 |
| 每文件数据行数 | 163,840 行 |
| 总数据量 | 约 3,932,160 行（~393 万点） |
| 通道数 | 4 通道（X / Y / Z / Sound） |
| 每文件采样时长 | ~6.4 秒 |
| 数据集大小 | ~224 MB |

## 7. 使用注意

1. **文件头部解析**：`skiprows=19` 跳过元数据，数据区以 `Time (seconds) and Data Channels` 行标记起始
2. **分隔符**：数据列之间使用**制表符（Tab，`\t`）**，非逗号
3. **数据单位**：原始数据为电压值（Volts），单位转换系数见头部 `Volts/Unit` 字段。本项目直接使用原始电压值进行特征提取，无量纲特征对单位不敏感
4. **数据平衡性**：6 种状态 × 4 种工况，每个文件大小一致，数据集完全平衡，无需类别不平衡处理
5. **数据划分**：建议混合所有文件进行分层随机划分，本项目采用 `StratifiedShuffleSplit` 8:2 划分
